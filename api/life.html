final case class World(liveCells: Cells, maybeBounds: Option[Bounds], maybeViewport: Option[Bounds])

A [World] on which the Game of Life will be played out.

A [World] on which the Game of Life will be played out.

Attributes

liveCells

The population of live [Cell]s in the current [World].

maybeBounds

The optional [Bounds] placed on the current [World]. If provided, the edges of the [World] are void, and any [Cell]s 'born' into the edge immediately fall off the edge. If not provided then the [Bounds] are endless.

maybeViewport

The optional [Bounds] for the view of the [World] when displaying. If provided, the [World] is only displayed within the range of the viewport's rows and columns. If not provided then the full [World] is displayed.
