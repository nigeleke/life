Generations

life.Generations
final case class Generations(world: World) extends Iterator[World]

An [Iterator] to provide each generation from an initial [World]. The iterations will stop when the [World] has repeated itself. Note: On an unbounded board, with a glider, the iterations will continue ad infinitum.

Value parameters

world

The initial [World].
